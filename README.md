# Netflix Content Strategy & Market Analysis

## Overview
This project involves a comprehensive analysis of Netflix's content catalog, comprmising over 8,800 titles added between 2008 and 2021. Using PostgreSQL for data extraction and Pyhton for visualization, we investigate how Netflix has evolved from US centric DVD rental service into a global streaming powerhouse. The analysis uncovers trends in content production, internal expansion and genre diversification, offering data driven insights into the company's strategic pivot towards 'Originals' and international markets.

## Goals
The primary objective of this analysis are divided into technical mastery and business intelligence

### Business & Analytics

- **Identify Content Trend:** Determine how the ratio of Movies to TV Shows has shifted over the last decade.
- **Analyze Global Expansion:** Map the rise of non-US content producers (e.g, South Korea, Turkey, UK) to understand Netflix's globalization strategy.
- **Target Audience Profiling:** Assess the maturity rating distrubution to understand the platform's core demographic.
- **Optimization Insights:** Discover seasonality patterns in content release (e.g,'Do they release more content in Q4?') to inform marketing strategies.
  
### Technical Goals

- **Advanced SQL Implementation:** Demonstrate proficieny in complex SQL operations, including:
  - 1. Handling arrays and delimeters
  - 2. Date and time manupilation
  - 3. Aggregation and grouping
  - 4. String parsing  

- **Data Visualization:** Utilizing Python to transform raw query outputs into actionable, boardroom ready charts.
